Lists, Tuples & Sets

Python offers several ways to group data together. The choice depends on whether you need the data to be ordered, unique, or changeable.

Structure Syntax Mutable? Description
List [1, 2, 3] Yes Ordered, allows duplicates.
Tuple (1, 2, 3) No Ordered, immutable (fixed).
Set {1, 2, 3} Yes Unordered, unique items only.

List Slicing

Extracting specific parts of a list using [start:stop:step].

nums = [10, 20, 30, 40, 50]
print(nums[0:2])  # [10, 20]
print(nums[::-1]) # [50, 40, 30, 20, 10] (Reverse)

Dictionaries & Key-Value Logic

A Dictionary stores data in Key: Value pairs. It is extremely fast for looking up information.

student = {
    "name": "John Doe",
    "course": "Python Basics",
    "grade": "A"
}

print(student["name"]) # John Doe
Unique Keys: Keys must be unique and immutable (like strings or numbers). Values can be anything, even other dictionaries!

List Comprehensions

A concise way to create lists. It combines a for loop and an optional if statement into one line.

# Traditional Way
squares = []
for x in range(5):
    squares.append(x**2)

# List Comprehension Way
squares = [x**2 for x in range(5)]

Filtering with Comprehensions

# Get only even numbers
evens = [x for x in range(10) if x % 2 == 0]
