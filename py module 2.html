Conditional Statements

Python uses if, elif, and else to execute code only when certain conditions are met.

age = 18

if age >= 18:
    print("You are an adult.")
elif age > 12:
    print("You are a teenager.")
else:
    print("You are a child.")
Indentation is Key! Unlike C or Java which use {}, Python uses 4 spaces (indentation) to define a block of code.

Repeating Tasks: Loops

1. The For Loop

Used for iterating over a sequence (like a list, string, or range).

# Prints 0 to 4
for i in range(5):
    print(f"Number: {i}")

2. The While Loop

Repeats as long as a condition remains True.

count = 0
while count < 3:
    print("Hello!")
    count += 1

Loop Control Statements

Sometimes you need to change how a loop behaves while it is running.

  • Break: Stops the loop entirely, even if the condition is still True.
  • Continue: Skips the rest of the current iteration and jumps to the next one.
  • Pass: A "do nothing" placeholder. Useful when you need a block of code syntactically but haven't written the logic yet.
for i in range(10):
    if i == 5:
        break  # Loop stops at 5
    if i % 2 == 0:
        continue  # Skips even numbers
    print(i)
