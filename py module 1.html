Installation & Package Management

Before coding, you need a Python interpreter and a way to manage libraries.

1. Pip (Python Package Index)

The standard package manager for Python. Use it to install external libraries.

pip install requests

2. Conda (Anaconda/Miniconda)

A powerful environment manager used primarily in Data Science. It allows you to create isolated "containers" for different projects to avoid version conflicts.

conda create --name my-project python=3.10
conda activate my-project
Why use environments? They prevent "Dependency Hell"—where one project requires Version A of a library and another project requires Version B.

Variables & Data Types

Python is dynamically typed, meaning you don't need to declare a variable's type explicitly.

Data Type Example Description
int age = 25 Whole numbers
float price = 99.99 Decimal numbers
str name = "Alice" Text characters
bool is_valid = True Logical True/False

Type Casting

Converting one data type to another manually.

x = "10"       # This is a string
y = int(x)     # Now y is an integer 10

Input, Output & String Formatting

Handling User Input

The input() function always returns a string.

name = input("Enter your name: ")
age = int(input("Enter your age: ")) # Casting to int immediately

String Formatting (f-strings)

The modern and most readable way to format strings in Python 3.6+.

name = "John"
score = 95
print(f"Hello {name}, your score is {score}%")

String Methods

  • .upper() - Converts to uppercase.
  • .strip() - Removes leading/trailing whitespace.
  • .replace("old", "new") - Swaps text segments.
