Defining Functions & Arguments

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function.

def greet_user(name, message="Welcome"):
    return f"{message}, {name}!"

# Calling the function
print(greet_user("Alice"))
print(greet_user("Bob", "Good morning"))

Types of Arguments:

  • Positional: Arguments that must be passed in the correct order.
  • Default: Arguments that take a value if none is provided (like message above).
  • *args: Allows a function to take any number of positional arguments.
  • **kwargs: Allows a function to take any number of keyword arguments (dictionary-style).

Lambda Functions & Variable Scope

Lambda (Anonymous) Functions

Small, one-line functions defined without a name. They are used for simple logic.

# syntax: lambda arguments : expression
square = lambda x : x * x
print(square(5)) # 25

Variable Scope: Local vs Global

Where a variable is defined determines where it can be accessed.

x = 10 # Global Scope

def my_func():
    y = 5 # Local Scope
    print(x + y) # Works!
The "Global" Keyword: To modify a global variable inside a function, you must declare it using global x inside that function.

Importing Standard Modules

Python comes with "batteries included," meaning it has many built-in modules ready for use.

1. Math Module

For mathematical constants and functions.

import math
print(math.sqrt(16)) # 4.0
print(math.pi) # 3.1415...

2. Random Module

For generating random numbers and choices.

import random
print(random.randint(1, 10)) # Random number between 1-10
print(random.choice(['Red', 'Green', 'Blue']))

3. OS Module

For interacting with the operating system and file folders.

import os
print(os.getcwd()) # Get current working directory
os.mkdir("NewFolder") # Creates a new folder
